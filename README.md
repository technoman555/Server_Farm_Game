## Goals

- Create a satisfying idle / management / tycoon-style game focused on building out increasingly ridiculous server farms
- Nail the aesthetic and feeling of real-world home labs, small colo setups, growing to hyperscale data centers
- Include fun nerd references (old hardware, RAID jokes, cooling horror stories, power bill panic, etc.)
- Make something we all actually want to play ourselves
- Learn a ton about game development along the way (especially in Godot!)
- Build a kind, collaborative, low-ego open-source game dev community around it

No pressure to ship fast â€” this is a long-term fun project.
